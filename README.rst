=========
PDFMerger
=========

|build| |codestyle|

Allows to merge pdfs without hassel.

Deployed at https://pdfmerger.vercel.app/

This doesn't save the uploaded/merged pdfs for long, uses ``tempfile`` and ``weakref`` modules.
